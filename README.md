# proyek-2-golang
Sistem Informasi Peminjaman Sarana dan Prasarana Kampus

Repository ini adalah backend untuk Sistem Informasi Peminjaman Sarana dan Prasarana Kampus.
yang dibangun menggunakan Bahasa Pemrogamman gGolang dan menggunakan Supabase (PostgreSQL) sebagai database utama.
Backend menyediakan layanan API untuk data ruangan, peminjaman, verifikasi, dan proses administrasi lainnya.
